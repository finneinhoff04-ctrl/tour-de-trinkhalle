SETUP
Nur du als Finn machst das – einmal, fertig.
1
Account erstellen

Geh auf supabase.com → "Start your project" → kostenlos registrieren.

2
Neues Projekt

„New project" → Name: finns-tour → Region: EU (Frankfurt) → Create.

3
Tabellen anlegen

SQL Editor → „New query" → einfügen und Run:

create table teams (
  id bigserial primary key,
  code text unique not null,
  name text not null,
  members jsonb default '[]',
  route text not null,
  points integer default 0,
  completed_stations jsonb default '[]',
  tasks jsonb default '{}',
  registered_at bigint
);

create table submissions (
  id bigint primary key,
  team_code text not null,
  station_id text not null,
  type text not null,
  submitted_at bigint,
  status text default 'approved',
  points integer default 0,
  note text default ''
);

alter table teams enable row level security;
alter table submissions enable row level security;

create policy "open" on teams for all using (true) with check (true);
create policy "open" on submissions for all using (true) with check (true);
4
API Keys holen

Project Settings → API → Project URL + anon public key kopieren.
